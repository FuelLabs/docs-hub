## Karşı sözleşme oluşturma

Sway'de bir sayaç sözleşmesi oluşturmak için aşağıdaki komutu çalıştırın:

```sh
forc template --template-name karşı sözleşme
```

Sözleşme `src/main.sw` dosyası içinde olacaktır.

## Sözleşmenin oluşturulması

Bir sözleşme oluşturmak için `counter-contract` klasörünün içine gidin:

``sh
cd karşı sözleşmesi
```

Ardından, `forc build` komutunu çalıştırın:

```sh
forc yapı
```
