# Getting Started with Fuel Network

Welcome to Fuel Network! This guide will help you get started as a user on Fuel Network, from setting up your wallet to exploring the ecosystem.

For more ecosystem dapps, please visit our ecosystem page [here](https://app.fuel.network/ecosystem).

## Network Information

### Mainnet (Fuel Ignition)

- Network Name: Fuel Ignition
- [RPCs](https://docs.fuel.network/docs/graphql/overview/#rpc-endpoints)
- [Explorer](https://app.fuel.network/)
- Chain ID: 9889
- Gas Symbol: ETH

### Testnet

- Network Name: Fuel Testnet
- [RPCs](https://docs.fuel.network/docs/graphql/overview/#rpc-endpoints)
- [Explorer](https://app-testnet.fuel.network/)
- Chain ID: 0
- Gas Symbol: ETH
