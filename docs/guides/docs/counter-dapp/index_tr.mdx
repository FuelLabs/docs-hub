# Sayaç Dapp

Bu kılavuz, aşağıdakilerin nasıl yapılacağına ilişkin adım adım talimatlar içerir

- Sway'de bir karşı akıllı sözleşme yazın
- Rust dilinde bir test yazın
- Fuel'in test ağına dağıtın
- Bir ön uç oluşturun
- Bir cüzdan entegre edin

Başlamadan önce, dokümanlar boyunca kullanılacak terminolojiyi ve bunların birbirleriyle nasıl ilişkili olduğunu anlamak faydalı olabilir:

- **Fuel**: Fuel blok zinciri.
- **FuelVM**: Fuel'e güç veren sanal makine.
- **Sway**: FuelVM için hazırlanmış alana özgü dil; Rust'tan esinlenilmiştir.
- **Forc**: Rust için Cargo'ya benzer şekilde Sway için yapı sistemi ve paket yöneticisi.
