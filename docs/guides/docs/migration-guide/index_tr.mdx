# Geçiş Rehberi

Fuel'in geliştirme yolculuğu boyunca, ana ağ lansmanına sorunsuz bir geçiş sağlamak için çok sayıda test ağı oluşturuldu.
Bu test aşamaları sırasında öğrenme ve uyarlamanın dinamik doğası göz önüne alındığında, kırılma değişiklikleriyle karşılaşmak yaygındır.

## Kırılma Değişiklikleri

Fuel araç zincirinin ve SDK'ların en son sürümlerine nasıl geçeceğinizi [Breaking Changes Log] (/guides/migration-guide/breaking-changes-log) bölümünden öğrenin.
